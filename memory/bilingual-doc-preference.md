When the user asks for a 中英双语 / bilingual document, they mean **one complete Chinese version and one complete English version as separate, self-contained files** — NOT Chinese prose with English terms interleaved in a single file.

**Why:** In this session the first draft used interleaved bilingual style (中文叙述 + 英文术语标注); the user corrected: "中英双语的意思是中文版和英文版各一套."

**How to apply:** Produce two files, e.g. `<name>.zh.md` and `<name>.en.md`, with fully parallel structure, numbering, and equations, each independently readable. Confirm file layout (two files vs. one file two sections) if ambiguous. See [[prove-repo]].
