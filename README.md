## 约定 / Conventions

- **双语 = 两套独立文件** / **Bilingual = two separate files**：每个主题给出完整中文版
  `*.zh.md` 与完整英文版 `*.en.md`，结构、编号、公式一一对应，各自可独立阅读。
- **数学 / Math**：使用 KaTeX 兼容的 LaTeX（`$...$`、`$$...$$`）。
- **查看 / Viewing**：任何支持 KaTeX 的 Markdown 阅读器（VS Code 预览、Obsidian、GitHub）。

无需构建或依赖，纯 Markdown 文档。
No build step or dependencies — plain Markdown documents.
